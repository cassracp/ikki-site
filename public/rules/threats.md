# Ameaças e Dano para Personagens Não Jogadores (PNJ)

| Nível de Ameaça | Dano Sugerido | Descrição da Ameaça |                              Exemplos                               |
| :-------------: | :-----------: | :-----------------: | :-----------------------------------------------------------------: |
|       0-1       |      d4       |     Comum/Baixa     | Sem treinamento, equipamento improvisado, animais selvagens menores |
|       2-4       |      d6       |    Padrão/Média     | Soldados treinados, armas de qualidade, vantagem numérica ou tática |
|       5-7       |      d8       |   Formidável/Alta   | Mestres em suas artes, bestas grandes, seres sobrenaturais menores. |
|       8+        |    d10/d12    |   Elite/Superior    |          Poderes sobrenaturais, armas de cerco/fogo, etc.           |

## Para Personagens de Mestre/Desafiante (PdM)

O Desafiante não rola ataque, os PJs rolam a defesa.

**Defesa:** 1d20 <= Mentsu +/- Bônus/Penalidade/Ameaça

Se o teste de defesa falhar, o PdM causa dano ao **Hara** do PJ.

Esse dano é arbitrado pelo desafiante conforme a ameaça/armas utilizadas pelo PdM. Mas, se o desafiante preferir, ele pode optar por utilizar a seguinte tabela como parametro:

| Ameaça |  Dano  |
| :----: | :----: |
|   1    |  1d4   |
|   2    | 1d4+1  |
|   3    |  1d6   |
|   4    | 1d6+1  |
|   5    |  1d8   |
|   6    | 1d8+1  |
|   7    |  1d10  |
|   8    | 1d10+1 |
|   9    |  1d12  |
|   10   | 1d12+1 |

_Nota:_ Um PJ com Hara zero recebe dano direto na carne/corpo ocasiando a morte, mutilação ou ferimento grave/hemorragia a ser arbitrada pelo diálogo ficcional ou rolagem de efeito do dano.

Defina a Ameaça do PdM com base na descrição. Por definição, a cada descritor do PdM ele causa +1 de ameaça.

Exemplos:

**Um bandoleiro usando uma lança.**

- Bandoleiros normalmente não são treinados, são bandidos comuns +0 de ameaça.
- Usando uma lança, ele gera +1 de ameaça.
- Total: +1 de ameaça.

**Um soldado usando uma espada.**

- Soldados normalmente são treinados, +1 de ameaça.
- Usando uma espada, ele gera +1 de ameaça.
- Total: +2 de ameaça.

**Um samurai usando armadura completa e daisho (Katana + Wakizashi).**

- Samurai são guerreiros de elite, +2 de ameaça apenas pelo treinamento.
- Usando um daisho, ele gera +2 de ameaça.
- Armadura completa, pode gerar +1 de ameaça ou 1 ponto de Resistência/Postura.
- Total: +4 de ameaça.
